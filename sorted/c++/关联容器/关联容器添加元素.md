# 添加元素

- 对于insert和emplace，添加单一元素, 返回一个[[pair]]对象
  - pair.first是一个指向给定键的迭代器
  - pair.second是一个bool值，插入成功返回True，失败返回false
  - 键存在, 插入失败，键不存在插入成功
- map添加元素
  - `m.insert({k, v});`
  - `m.insert(make_pair(k, v));`
  - `m.insert(pair<string, size_t>(s, val));`
  - `m.insert(map<string, size_t>::value_type(s, val))`
