# ideavim配置

点击ideavim插件图标可以修改`~/.ideavimrc`配置文件

- 当前(20211020)配置文件内容

```vim
set scrolloff=5  
  
set incsearch   
set hlsearch        " 高亮搜索
set relativenumber  " 相对行号
set nu  
set ts = 4  
set expandtab  
  
  
map Q gq  
map <CapLocks> <ESC>  
nmap <C-j> 10j
vmap <C-j> 10j  
nmap <C-k> 10k  
vmap <C-k> 10k  
```
