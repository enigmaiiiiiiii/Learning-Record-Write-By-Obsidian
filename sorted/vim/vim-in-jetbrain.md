# Ideavim Config

click ideavim plugin icon and edit the  `~/.ideavimrc` config

- current(20211020) .ideavimrc

```vim
set scrolloff=5

set incsearch
set hlsearch        " high light search result
set relativenumber  " relative line number
set nu
set ts = 4
set expandtab

map Q gq
map <CapLocks> <ESC>
nmap <C-j> 10j
vmap <C-j> 10j
nmap <C-k> 10k
vmap <C-k> 10k
```
