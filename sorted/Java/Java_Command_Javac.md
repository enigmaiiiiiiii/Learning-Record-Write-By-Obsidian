# javac: java源文件编译器

```shell
javac [options] [sourcefiles-or-classnames]
```

参数: sourcefiles-or-classnames

- sourcefiles: 编译的源文件
- classnames: 之前编译完成的类文件

大量文件时, 使用@filename指定文件参数，文件内容为源文件列表
