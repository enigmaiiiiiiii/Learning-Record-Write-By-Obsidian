## 典型项目结构

- `<name>-sso`: 单点登录
  - Spring Security相关
  - 发放token
  - 解析token
- `<name>-common`: 公共模块
- `<name>-search`: 搜索模块
- `<name>-pojo`: 实体类
- `<name>-search`: 搜索模块
- `<name>-gateway`: 网关
- ...
