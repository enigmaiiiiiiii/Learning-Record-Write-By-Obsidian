# Elasticsearch

[documentation](https://www.elastic.co/guide/cn/elasticsearch/guide/current/running-elasticsearch.html)

- 用来代替mysql数据库的模糊搜索功能

## 启动

```bash
cd elasticSearch-<version> # 进入安装目录
./bin/elasticsearch # linux启动
./bin/elasticsearch.bat # windows启动
```
