# 原型(archetype)

Maven QuickStart Archetype

```shell
project
│  pom.xml
│
├─src
│  ├─main
│  │  └─java
│  │      └─com
│  │          └─example
│  │                  App.java
│  │
│  └─test
│      └─java
│          └─com
│              └─example
│                      AppTest.java
│
└─target
    ├─classes
    │  └─com
    │      └─example
    │              App.class
    │
    └─test-classes
        └─com
            └─example
                    AppTest.class
```
