# 创建String对象

- 编译时创建字符串对象
  - `"hello"`: 字符串字面量
  - `"hel" + "lo"`: 通过`+`连接字符串字面量得到的运算后的对象
- 运行时创建
  - `str + "hello";`: 变量 + 字面量
  - `String str = new String(StrVal);`

## 字面量

- 隐式创建String类的新实例, 并保存在字符串池中, 如果之前没有相同的字符串字面量

## 字符串池

- 由String类维护
- 方法`("hello").intern()`返回字符串池中的`"hello"`对象
  - 池中已经存在，则返回该对象
  - 池中不存在，则在池中添加该对象
