# 数组

- 是一种[引用类型]
- 元素类型必须相同
- 必须指定元素个数
- 数组长度不可改变
- 初始值为0, 0.0, false

## 创建数组

```java
int[] a1 = new int[5];               // 创建一个长度为5的数组, 元素值为0
int[] a2 = {1, 2, 3, 4, 5};          // 创建一个长度为5的数组, 元素值为1, 2, 3, 4, 5
int[] a3 = new int[]{1, 2, 3, 4, 5}; // 创建一个长度为5的数组, 元素值为1, 2, 3, 4, 5
```

## 元素

- 数组元素的值为地址, 引用对象实例

## 数组复制

- 复制指定数组，截断或填充

```java
Arrays.copyOf(T[] original, int newLength)
```
