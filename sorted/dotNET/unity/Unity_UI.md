# UI

- 三个UI系统
  - UI toolkit
  - Unity UI(uGUI)
  - IMGUI

## [UI工具包](Unity_Script_UI_toolkit.md)

- 最新的UI系统, 功能不完善
- 基于标准Web技术(如HTML, XML, CSS)
- 可以用于
  - InspectorUI
  - 编辑器扩展
  - 应用程序调试
  - 应用程序运行时UI

## [Unity UI](Unity_UI_uGUI.md)

- 可以用于
  - 应用程序运行时UI
- 不能用于创建编辑器UI  

## IMGUI

- 可以用于
  - InspectorUI扩展
  - 编辑器扩展
  - 应用程序调试UI
