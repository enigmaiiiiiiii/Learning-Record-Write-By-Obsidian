# 集合

- 需要实现遍历的集合通过一些列[接口](csharp_interface.md)来支持该需求

## 泛型集合

- System.Collections.Generic类
- 包括`Dictionary<TKey, TValue>`, `List<T>`, `Queue<T>`, `SortedList<TKEY, TValue>`, `stack<T>`

## 所有元素存储为object类型的集合

- System.Collections类
- ArrayList, Hashtable, Queue, Stack
