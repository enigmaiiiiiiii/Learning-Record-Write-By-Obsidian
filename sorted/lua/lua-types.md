
## Array

**By Default: lua start array index from 1**

```lua
a = {1, 2, 3}
print(a[1]) -- 1
```

array index can start at any number

```lua
a = {}
for i=-5, 5 do
  a[i] = 0
end
```

New Array

```lua
a = {} -- empty array
```
