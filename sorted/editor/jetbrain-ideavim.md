# Ideavim

```shell
# 在jetbrain中设置easymotion插件
Plug 'easy-motion/vim-easymotion'
set easymotion
map <leader><leader> <Plug>(easymotion-prefix)
set scrolloff=5


# highlight search, ignore case
set incsearch
set hlsearch
set ignorecase
set nu
set ts = 4
set expandtab

# 键盘映射
map Q gq
nmap ; $
nmap <C-j> 10j
vmap <C-j> 10j
nmap <C-k> 10k
vmap <C-k> 10k
```
