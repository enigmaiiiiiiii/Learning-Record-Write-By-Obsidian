## 操作

 操作是函数声明，方法是具体函数

- 语法

```
visibility name (parameter-list) : return type {property-string}
```

- visibility: 公有(+), 私有(-)
- name: 方法名称
- parameter-list: 参数列表

参数列表中的表示方法

  ```
  direction name: type = default value
  ```

direction是指参数是输入(in)，输出(out)，还是输入输出(inout)  
 
- return-type: 返回类型
