# 时间复杂度分析

- 用$O(f(n))$来表示时间复杂度
- 假设$T(n)$是数据规模为$n$时所消耗的时间，  
如果存在整的常数$C$和$n_0$,使得当$n \geq n_0$时都满足$0 \leq T(n) \leq C*f(n)$,  
则称$T(n)$与$f(n)$同数量级，并记作$T(n) = O(f(n))$为该算法的时间复杂度。

> $T(n) = 2n^3 + 3n^2 + 2n + 1$  
  当$n \rightarrow \infty$时：  
  $T(n) / n^3 = (2n^3 + 3n^2 + 2n + 1) / n^3 \rightarrow 2$  
  所以时间复杂度时$O(n^3)$

- 常见的时间复杂度
  - $O(1)$ 
  - $O(log_2n)$
  - $O(n)$
  - $O(n^2)$
  - $O(n^3)$

- 分析方法
  - 选取一种基本操作分析, 该操作的次数
- 通常以最坏的时间复杂度作为算法的复杂度
