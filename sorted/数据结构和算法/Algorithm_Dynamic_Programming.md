# 动态规划

[[leetcode322]]

- 动态规划问题中, 每个值由之前的过程递推得到

## 动态规划的关键问题
- 递推公式
- dp数组
  - dp数组的初始化
  - dp数组下标的含义
  - dp数组的元素一般为所求结果
  - 是否需要遍历dp数组

## 01背包

- N件物品，第i建物品的重量为`w[i]`, 价值为`v[i]`
- 状态数组`dp[i][j]`的含义: 前i件物品装进限重为j的背包可以获得的最大价值

```java
dp[i][j] = dp[i - 1][j];
dp[i][j] = dp[i - 1][j - w[i]] + v[i];
```
