# ARP协议

- 实现任意网络层地址到任意物理地址的转换

## ARP协议工作原理

- 地址解析协议
- 主机向自己所在的网络广播一个ARP请求  
   请求包含目标机器的网络地址
   此网络上的所有机器都将受到这个请求
   但只有被请求的机器会回应一个ARP应答
   其中包含自己的物理地址
- 以太网ARP请求/应答报文格式

- ARP维护一个高速缓存，其中宝海经常访问或最近访问的机器的IP地址到物理地址的映射, 避免了重复ARP请求，提高了发送数据包的速度
