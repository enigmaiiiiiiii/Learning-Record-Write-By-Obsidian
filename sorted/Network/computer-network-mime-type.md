# Computer Network - MIME Type

- MIME (Multipurpose Internet Mail Extensions)
- A standardized way of describing file type
- consist of two parts
  - type
  - subtype

| type/subtype     | extension name |
| ---------------- | -------------- |
| application/pdf  | .pdf           |
| text/plain       |                |
| text/html        | .html .htm     |
| image/jpeg       | .jpg .jpeg     |
| application/json | .json          |
