# Vue组件

- 组件是可复用的[Vue实例](Vue_Instance.md)
- HTML中表现为自定义元素, 如`<my-component />`
- 包括data, computed, watch, methods, 生命周期钩子等
  - 不同的是, data必须是函数, 保证每个组件实例都有独立的data 
- 不包括el
