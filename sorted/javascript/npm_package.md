## npm作用域

- 当你创建npm**用户账号**或创建一个**组织**时，你就获得了一个相应的作用域
- 作用域中的package名称允许与其它用户有相同
- 当package作为依赖在[package.json]文件中列出时, `@scopename/package-name`, scope名称位于`@`和`/`之间
