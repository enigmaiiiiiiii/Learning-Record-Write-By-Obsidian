# three.js Object3D

## what is this

- base class for all objects in three.js
- mostly use as a root of a model

```
root
 ├─*no-name*[CameraHelper]
 ├─*no-name*[CameraHelper]
 ├─cameraRig[Group]
 | ├─*no-name*[PerspectiveCamera]
 | ├─*no-name*[OrthographicCamera]
 | └─mesh3[Mesh]
 ├─*no-name*[Mesh]
 | └─*no-name*[Mesh]
 └─*no-name*[Points]
```
