## 安装

Linux

```shell
sudo apt intall nvm  # 安装Node版本管理工具nvm(Node Version Manager)
nvm install node  # 安装node最新可用版本 
nvm ls-remote  # 列出远程可安装的版本 
```

> nvm: Node Version Manager
