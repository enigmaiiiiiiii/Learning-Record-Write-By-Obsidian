# NPM - Scoped Package

## Install A Scoped Package

```json
{
    "dependencies": {
        "@myorg/mypackage": "^1.3.0"
    }
}
```

## Pubish A Scope Package

publishing to your user scope

```json
{
    "name": "@username/mypackage",
    "version": "1.0.0"
}
```

publishing to organizatoin scope

```json
{
    "name": "@org/mypackage",
    "version": "1.0.0"
}
```
