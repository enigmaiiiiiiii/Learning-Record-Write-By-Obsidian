# Iteration in JavaScript

- **调用**生成器函数, 会返回一个[Generator](javascript-generator.md)对象
  - 生成器对象开始时处于暂停状态, 也就是说函数体内的代码还没有执行
  - 生成器函数在首次调用`next()`方法时, 开始执行
- 通过调用生成器对象的next()方法
- yield关键字
