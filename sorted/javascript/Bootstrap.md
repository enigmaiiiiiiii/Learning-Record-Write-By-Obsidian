# Bootstrap

## How to use

[Next.js](nextjs-css-support.md#from-node-modules)

`pages/_app.js`

```js
import `bootstrap/dist/css/bootstrap.css`

export default function App() {
  return (
    <div className="App">
      <h1>React Bootstrap</h1>
      <Button variant="primary">Primary</Button>
    </div>
  );
}
```
