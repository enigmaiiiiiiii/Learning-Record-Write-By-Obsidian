# Object

- 所有引用类型继承自Object

## 方法

- toString(): 返回表示该对象的字符串
- valueOf(): 返回对象的原始值
- isPrototypeOf()
- create()
  - 创建一个新对象, 使用现有对象作为新对象的原型


## Static Method

`Object.keys(obj)`

- 返回一个数组, 包含对象的所有可枚举属性的名称

`Object.values(obj)`

- return an **array** containing the given object's own [enumerable](javascript-property-sort.md#data-property) string-keyed property values
