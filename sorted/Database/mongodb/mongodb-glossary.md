# Glossary

## document

- composed of field and value

```js
{
  name: "sue",
  age: 26,
  status: "A",
  groups: [ "news", "sports" ]
}
```

## collection

- store [documents](#document)

## Databases

- hold one or more collection

## Field

- key in a document

## mongod

- primary **daemon process** for the MongoDB

## namespace

- canonical name for a collection
- like `[database-name].[collection-or-index-name]`
