# Blender Operate

## Switch Mode

- tab

## Object Mode

- g: grab
- gx: grab and move on x axis
- r: rotate
- rx: rotate on x axis
- s: scale
- sx: scale on x axis
- `/`: local view or focus on select

## Edit Mode

- g: grab
- alt + s: flat translate select
- crtl + l: select linked part
- e: extrude

## Useful Panel

- `z`: object shading

## Glossary

- subvision

## Moving

- snap moving: shift + tab
