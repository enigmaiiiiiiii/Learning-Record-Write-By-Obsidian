# blender

## Switch Mode

tab

## Object Mode

- g: grab
- gx: grab and move on x axis
- r: rotate
- rx: rotate on x axis
- s: scale
- sx: scale on x axis

## Edit Mode

- g: grab
- alt + s: flat translate select
- crtl + l: select linked part

## Useful Panel

- `z`: object shading
