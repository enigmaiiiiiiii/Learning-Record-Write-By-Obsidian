# 启动工作

1. 连接

```shell
mysql -u root -p
```

2. 创建数据库

```sql
CREATE DATABASE 数据库名称
```

3. 连接数据库

```sql
USE 数据库名称 
```

4. 创建数据表

```sql
CREATE TABLE name(列名称 列类型)
```

5. 插入数据

```
INSERT INTO table_name(filed1, field2, ..., fieldN)
                       VALUES
                       (val1, val2, ..., valN);
```

6. 查询数据

```
SELECT * FROM table_name;
```
