# Docker - Start To Use

## search image

```sh
docker search <name>
``
```

## Start A Container

- start containers is a complex operate
- need to set up name, port, volume, network, etc
- check [docker run](docker-command-run.md)

## download image

```sh
docker pull <name>
```

## check images

```sh
docker images
```

## check running container

```sh
docker ps
```
